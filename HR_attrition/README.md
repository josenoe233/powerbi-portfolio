## Objective
Analyze employee attrition to identify key drivers such as age, salary, job role and tenure.

## Dataset
HR Analytics dataset with employee demographics, performance and attrition indicators.

## Data Model
Star-schema inspired model with FactEmployees as the central table.

## Key Metrics
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary

## Key Insights
- Highest attrition occurs in early career stages
- Sales roles show higher turnover
- Lower salary bands have higher attrition

- Power BI
- Power Query
- DAX
